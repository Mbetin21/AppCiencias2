# Carpeta de Imágenes

Esta carpeta contiene los recursos visuales de la aplicación.

## Imágenes disponibles

### hamburger.png
- **Uso:** Icono del botón de menú lateral (hamburger menu)
- **Tamaño recomendado:** 24x24 píxeles (se escalará automáticamente)
- **Formato:** PNG con transparencia
- **Ubicación:** `images/hamburger.png`

## Instrucciones

1. Coloca tu imagen en esta carpeta con el nombre correcto
2. La aplicación cargará automáticamente la imagen al iniciar
3. Si la imagen no existe, se mostrará el símbolo de texto "☰" por defecto

## Formatos soportados

- PNG (recomendado para transparencia)
- JPG
- GIF

## Notas

- Asegúrate de que las imágenes no sean muy pesadas para mantener el rendimiento
- Los colores deben contrastar bien con el fondo gris claro (#F0F0F5)
- Se recomienda usar iconos con transparencia para mejor integración visual
